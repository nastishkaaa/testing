About the workshop

Creative Workshop Hub is a demo event website for students who want to improve practical skills in web design, accessibility testing, content creation and presentation.

The website contains many interactive parts: filters, cards, programme timeline, registration form, pricing selection, FAQ, modal windows and dynamic messages.

Important information: participants should bring their own laptop and charger.

What you will learn

  • How to plan and design a small web project.
  • How to test accessibility problems in real interfaces.
  • How to prepare a portfolio presentation.
  • How to work in a small creative team.
